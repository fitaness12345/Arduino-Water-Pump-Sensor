# Arduino-Water-Pump-Sensor
A simple water pump with sensor programmed with Arduino.

### Materials 

- Arduino Nano / Uno
- Water Pump
- Battery
- Sensor
- Single Channel Relay
- Data Cable
- Jumper Wires
